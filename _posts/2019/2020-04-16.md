### ios调起其他app报错
```
This app is not allowed to query for scheme cmbmobilebank
```

### 解决办法

修改Info.plist, 添加

```
<key>LSApplicationQueriesSchemes</key>
<array>
        <string>cmbmobilebank</string>
</array>
```
